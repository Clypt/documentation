## Before You Begin

ClyptAI connects to your exchange via API keys. This allows the platform to read your portfolio and execute trades on your behalf, without ever having access to withdraw your funds.

<Warning>
  **Never enable withdrawal permissions.** ClyptAI only requires read access and trade execution. If any setup guide asks you to enable withdrawals, stop and contact support immediately.
</Warning>

## Security Principles

All ClyptAI integrations follow these security standards:

- **IP Whitelisting** - API keys are restricted to ClyptAI server IPs only
- **No Withdrawal Access** - Keys are configured with trade and read permissions only
- **Encrypted Storage** - All API credentials are encrypted at rest with AES-256
- **Minimal Permissions** - Only the permissions required for trading are requested
