---

<Warning>
  When configuring API permissions, **never** enable "Enable Withdrawals". ClyptAI only requires read access and trade execution permissions.
</Warning>

<Steps>
  <Step title="Navigate to API Management">
    Log in to your Binance account. Click on your **Profile Icon** in the top right corner to access your main Dashboard.

    1. On the left sidebar menu, find the **Account** section, scroll down, and click **API Management**.
    2. Click the yellow **Create API** button located at the top right.
    3. A "Choose API Key type" pop-up will appear. Select **System generated** and click **Next**.
    4. Give your API key a label (e.g., "ClyptAI Trading") and complete the security verification to proceed.
  </Step>
  <Step title="Edit Restrictions">
    Binance requires you to bind trusted IP addresses before you can enable trading permissions.

    1. On your newly created API key block, click the yellow **Edit restrictions** button in the top right corner.
    2. Scroll down to the **IP access restriction** section at the bottom.
    3. Select **Restrict access to trusted IPs only (Recommended)**. An input box will appear.

    <Frame>
      ![Edit Restrictions](/images/Screenshot2026-03-04at5.13.52PM.png)
    </Frame>
  </Step>
  <Step title="Copy IP Whitelist from ClyptAI">
    Keep your Binance tab open.

    1. Open a new browser tab and log in to your **ClyptAI** dashboard.
    2. Navigate to the exchange connection page and select **Binance** from the dropdown menu.
    3. Locate the **IP Whitelist** box and click the **Copy** button.

    <Frame>
      ![ClyptAI IP Whitelist](/images/Screenshot2026-03-04at5.16.17PM.png)
    </Frame>
  </Step>
  <Step title="Paste IP and Configure Permissions">
    1. Go back to your **Binance** tab.
    2. Paste the copied IP whitelist into the IP restriction box.
    3. Click **Confirm** next to the input box. The IP addresses will be listed in yellow blocks.
    4. Scroll up slightly to the **API restrictions** section. Check the following permissions:

    - **Enable Reading**
    - **Enable Spot & Margin Trading**
    - **Enable Futures**

    <Frame>
      ![Permissions](/images/Screenshot2026-03-04at5.22.14PM.png)
    </Frame>

    <Warning>
      Do **not** check "Enable Withdrawals".
    </Warning>
  </Step>
  <Step title="Save Your API Key">
    Click the yellow **Save** button at the top right of the API block. You may need to enter your 2FA code again to save changes.

    1. Go back to your open **ClyptAI** tab.
    2. Copy the **API Key** from Binance and paste it into the **API Key** field in ClyptAI.
    3. Copy the **Secret Key** from Binance and paste it into the **API Secret** field in ClyptAI.
    4. Click the **Connect** button to finish.

    <Warning>
      Binance will only show your **Secret Key** once. If you navigate away or refresh the page, the Secret Key is lost permanently. Paste it into ClyptAI immediately.
    </Warning>
  </Step>
  <Step title="Verify the Connection">
    To confirm the integration is working:

    In the top right corner of the dashboard page, you will now see the **Connected Exchange** status.

    Your Binance account is now securely integrated and ready for automated execution. You can now start deploying your alpha.

    <Frame>
      ![Connected](/images/Screenshot2026-03-04at5.27.20PM.png)
    </Frame>
  </Step>
</Steps>
