---

<Warning>
  When configuring API permissions, **never** enable "Transfer" or withdrawal permissions. ClyptAI only requires read access and trade execution.
</Warning>

<Steps>
  <Step title="Navigate to API Settings">
    Log in to your Coinbase account. Click on your **Profile Icon** in the top right corner and select **Settings**.

    On the left sidebar menu, scroll down and click **API**.

    ![Coinbase Settings](/images/Screenshot2026-03-04at4.01.23PM.png)
  </Step>
  <Step title="Create API Key">
    Click the blue **Create API Key** button to start generating a new key for ClyptAI.

    ![Create API Key](/images/Screenshot2026-03-04at4.03.18PM.png)
  </Step>
  <Step title="Configure Permissions">
    In the "Create API Key" pop-up window, configure the following:

    1. **API Key Nickname** - Give your key a recognizable name (e.g., "ClyptAI").
    2. **Portfolio** - Leave it as "Primary".
    3. **Permissions** - Check the first two boxes:
       - **View** (Read-only)
       - **Trade** (Execute trades on your behalf)

    Do not click the create button yet. You need to restrict the IP first.

    ![Permissions](/images/Screenshot2026-03-04at4.08.09PM.png)
  </Step>
  <Step title="Copy IP Whitelist from ClyptAI">
    Before finalizing the key, retrieve the authorized IP addresses from ClyptAI.

    1. Open a new browser tab and log in to your **ClyptAI** dashboard.
    2. Navigate to the exchange connection page and select **Coinbase** from the dropdown menu.
    3. Locate the **IP whitelist** box and click the **Copy** button.

    <Frame>
      ![ClyptAI IP Whitelist](/images/Screenshot2026-03-04at4.11.13PM.png)
    </Frame>
  </Step>
  <Step title="Paste IP and Generate Key">
    1. Go back to your **Coinbase** tab.
    2. Paste the copied IP whitelist into the **IP allowlist** text box.
    3. Click the blue **Create & Download** button at the bottom. Complete any 2FA security checks if prompted.

    <Warning>
      Do **not** check the "Transfer (initiate transfer of funds)" permission. ClyptAI will never request withdrawal access.
    </Warning>

    <Frame>
      ![IP Allowlist](/images/Screenshot2026-03-04at4.15.29PM.png)
    </Frame>
  </Step>
  <Step title="Save Keys and Connect to ClyptAI">
    Coinbase will generate your API credentials. The keys will only be displayed **once**.

    <Info>
      **Immediately** copy your API Key and Secret Key. Paste them securely into your personal notes or directly into the ClyptAI connection screen.
    </Info>

    1. Go back to your open **ClyptAI** tab.
    2. Copy the **API Key name** from Coinbase and paste it into the **API Key** field in ClyptAI.
    3. Copy the **Private Key** from Coinbase and paste it into the **API Secret** field in ClyptAI.
    4. Click the **Connect** button to finish.
  </Step>
  <Step title="Verify the Connection">
    To confirm the integration is working:

    In the top right corner of the dashboard page, you will now see the **Connected Exchange** status.

    Your Coinbase account is now securely integrated and ready for automated execution. You can now start deploying your alpha.

    <Frame>
      ![Connected](/images/Screenshot2026-03-04at4.38.29PM.png)
    </Frame>
  </Step>
</Steps>
